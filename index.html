Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit 1.0

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • Github
  • Github
  • Dashboard

Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit 2.0

Este projeto desenvolveu um painel interativo utilizando Python e Streamlit para analisar dados de restaurantes da API Zomato, focando no mercado de delivery de comida na Índia. O painel oferece visualizações das principais métricas, como avaliações, tipos de culinária e custo médio, atualizadas semanalmente, permitindo insights sobre as tendências gastronômicas.

A ferramenta é útil para profissionais de marketing, donos de restaurantes e consumidores, proporcionando uma visão clara sobre o desempenho e as preferências culinárias nas diversas regiões, ajudando na tomada de decisões informadas sobre onde comer.

As ferramentas utilizadas foram:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • Github
  • Github
  • Dashboard

Respondendo perguntas de negócios utilizando SQL

O dataset de e-commerce brasileiro da Olist contém cerca de 100 mil pedidos realizados entre 2016 e 2018, abrangendo múltiplos marketplaces no Brasil. Ele permite análises detalhadas de pedidos com informações sobre status, preços, pagamentos, frete, localização dos clientes, atributos dos produtos e avaliações.

As análises realizadas incluem identificação de categorias e produtos mais vendidos ou valiosos, cálculos de médias móveis, rankings de preços e avaliações dos clientes, utilizando comandos SQL para responder a perguntas de negócios. O dataset é uma fonte rica para explorar o desempenho do comércio eletrônico no Brasil, com dados anonimizados e integrações possíveis com o funil de marketing da Olist.

As ferramentas utilizadas foram:

  • SQL
  • SQlite3
  • DBeaver
  • Github
  • Github
